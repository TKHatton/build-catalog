## How to un-hide a screenshot

In `README.md`, find the block:
```
<!-- Screenshot: uncomment once image is added to /screenshots
![StormIQ screenshot](screenshots/stormiq.png)
-->
```
Remove the `<!--` opening and `-->` closing lines, leaving just the `![...](...)` line.
